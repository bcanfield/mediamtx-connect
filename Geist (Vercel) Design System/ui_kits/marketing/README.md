# Marketing Site UI Kit

A full Vercel-style landing page recreated on the Geist system, composing the design system's own components (`NavBar`, `Hero`, `Card`, `CodeBlock`, `LogoStrip`, `CTABand`, `Footer`, `Button`, `Input`).

## Screens
- `index.html` — the complete home page: nav → mesh hero → logo strip → 3-up feature grid → zero-config deploy band with terminal → interactive template tabs (AI Apps / Web Apps / Ecommerce / Docs) → signup card → CTA band → footer.

## Files
- `MarketingPage.jsx` — the composed page and its local section helpers.

## Notes
- Interactive: the template category tabs switch content on click.
- No logo asset ships with this system, so the wordmark renders as tightly-tracked type.
- Color is confined to the hero mesh and the template thumbnails; everything else is ink on near-white.
