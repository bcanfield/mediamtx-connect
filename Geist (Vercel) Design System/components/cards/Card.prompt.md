The hairline content card that builds every Geist feature grid — flat by default, with optional eyebrow + title slots.

```jsx
<Card eyebrow="Edge" title="Fluid compute" >
  Pay only for what you use, down to the millisecond.
</Card>
<Card elevation={2} radius="lg" title="Pro" padding="var(--space-xl)">…</Card>
```
`elevation`: 0 flat (default), 1 whisper, 2 floating. `radius`: md (feature, 12px), lg (pricing, 16px), sm (6px).
