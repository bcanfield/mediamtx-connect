A single-line text field with optional label, hint, and error state — the default Geist form input.

```jsx
<Input label="Email" placeholder="you@example.com" />
<Input label="Team slug" hint="Lowercase, no spaces" />
<Input label="Email" error="Enter a valid email" defaultValue="nope" />
```
Border is a 1px hairline that darkens to ink on focus; the error state turns it red.
