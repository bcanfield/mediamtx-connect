The optimal-stack showcase pins typescript to ~5.9.3 instead of the TS 6.x that create-vite ships today, because openapi-typescript@7.13.0 declares a typescript@^5.x peer dependency and npm refuses the resolution. Forcing with --legacy-peer-deps would run the codegen against an unsupported compiler. Bump to TS 6 once openapi-ts widens its peer range; also noted in the showcase README under "Known constraints".
