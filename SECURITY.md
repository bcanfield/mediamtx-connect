# Security Policy

## Supported Versions

Only the latest released version of MediaMTX Connect is supported with security
updates. Please upgrade to the latest release before reporting a vulnerability.

## Reporting a Vulnerability

If you find a security vulnerability, please report it privately using GitHub's
private vulnerability reporting:

1. Go to the [Security tab](https://github.com/bcanfield/mediamtx-connect/security) of this repository.
2. Click **Report a vulnerability**.
3. Include as much detail as you can: affected version, reproduction steps, and
   potential impact.

**Please do not open a public issue for security reports.** This gives us a
chance to investigate and release a fix before the details are public.

## What to Expect

We'll acknowledge new reports on a best-effort basis, typically within a few
days, and keep you updated as we investigate and work on a fix. Timelines vary
depending on severity and complexity.
